installation proecdure 

 to install an run the project it is necessaery to setup speech recognization in our system.
after that install the WSR macros software to get the voice out and to run the xml script.

After setting up this two system thired wee need to upload the code into oue esp8266 
microcontroller. to do such arduino IDE is need to statrt and connect the esp8266 to computer 
using micro USB cable.

after installation our system is ready to perform. start the esp8266 make the connections with relay
and power up the system.
